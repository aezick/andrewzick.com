Computer Security


Twitter is a great place for peeking at what's on the mind of the security community, not to mention most of them are pretty nice. SwiftOnSecurity mixes in jokes with topical security commentary and often conversations they start will have lots of important security people from around the web chiming in.


If you're eager to jump into security but don't feel adequately prepared, check out The Security Setup. It's dedicated to interviewing people from the security community about what hardware/software they use and it's chock full of useful information, from easy changes like installing an adblocker (I use uBlock Origin) and enforcing HTTPS Everywhere.

Now that you're ready, the best way to dive into the field is to start completing CTFs, or Capture the Flags, special computer security events held to test how you apply security skills to computer problems. CTF Time does a better job explaining what kinds of challenges you'll see in CTFs and hosts a calendar with future CTFs that you can participate in. Here's a list of CTFs in no particular order:

  • Trail of Bits CTF Field Guide
  • OverTheWire Wargames
  • High School CTF 3 Practice Problems
  • HackThisSite
  • Pico CTF 2014
  • Praetorian's Tech Puzzles
  • The DEFCON CTF VM (old CTFs from DEFCON)

I also have a particular interest in password security. I use Hashcat and a modified version of the CrackStation wordlist for dictionary attacks on my own passwords for fun because if my CPU can crack it then I shouldn't be using it for a password! Also you should sign up for Have I Been Pwned? if you haven't already. It notifies you any time your email address appears in a company's database leak so you can go change the password right away. Here's a bunch of great resources on passwords:

  • CyLab Usable Privacy and Security Laboratory (academic research from Carnegie Mellon)
  • Password Incentives Based on Longer Time Before Expiration
  • Conference Talk about Password Complexity
  • In-depth Comparison of Password Generators
  • Test Your (Bad) Passwords Here!
